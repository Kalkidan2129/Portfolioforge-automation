# PortfolioForge AI

## Overview

PortfolioForge AI is an automated portfolio generation platform built with Node.js and Playwright. The system authenticates into the Colaberry platform, processes multiple project links, extracts structured project information, and automatically generates recruiter-friendly portfolio artifacts including README files and structured JSON data.

---

## Features

- Dynamic multi-project processing
- Authenticated browser automation
- Automated project data extraction
- Dynamic README.md generation
- Structured JSON artifact generation
- Reusable authenticated session handling
- Automated folder and file creation
- Safe fallback handling for missing project fields

---
